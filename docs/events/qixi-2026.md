# 七夕星橋願望祭 2026 製作紀錄

## 活動定位

- 活動日期：2026-08-19
- 活動名稱：七夕星橋願望祭
- 套裝名稱：星橋願望套裝
- 活動層級：節日主檔
- 接續活動：2026-08-27 中元夜市補給令，小型活動，不搶七夕主視覺
- 核心語氣：星河學院祭、鵲橋、願望帳本、溫暖財務冒險

## 製作硬規則

- 先產出完整男女雙版本角色設計，再做動畫，不用靜態圖假裝正式套裝。
- 女主角與男主角是同一活動主題，但髮型、剪影、表情、姿勢和手持物都要各自設計。
- 角色、服裝、髮型、髮飾、手持物必須在同一張角色主圖內完成，不拆成紙娃娃零件。
- 手持物必須被角色拿著或穿戴，不得漂浮在旁邊。
- 背景必須有可踩的橋面、屋頂或祭典平台，角色不能浮在星空上。
- 正式前端資產只收透明 Animated WebP 和壓縮後背景 WebP，不提交原始影片、臨時幀或失敗版本。

## 女主角方向

- 髮型：側編髮或雙辮，加入星形髮飾與月白挑染。
- 表情：溫柔、期待，像正在寫下願望。
- 服裝：星紗短斗篷加學院祭禮服，深靛、星術紫、月白和獎勵金為主。
- 手持物：願望帳本與羽毛筆，或星砂瓶。
- 動作：寫下願望後抬頭看星河，星砂瓶微微發光。

## 男主角方向

- 髮型：蓬鬆短髮或低馬尾，加入小型星圖髮飾或月白挑染。
- 表情：沉著、帶一點少年感的自信。
- 服裝：輕便浴衣式學院祭外套或星紗短披肩，剪影要和女主角不同。
- 手持物：鵲橋票券、小型星盤或願望帳本。
- 動作：牽引星線或遞出鵲橋票券，腳底穩定不漂移。

## 背景方向

- 場景：學院屋頂或鵲橋祭典平台。
- 近景：可踩的橋面、石階、木平台或屋頂邊緣。
- 中景：願望籤、星燈、帳本祭小攤，保留二至三個必要線索。
- 遠景：銀河、遠處鵲橋、柔和星線。
- 禁止：角色、文字、UI、logo、浮水印、大量粉紅愛心粒子。

## 小動畫方向

- 星塵緩慢閃爍。
- 願望籤輕擺。
- 遠方星線低速流動。
- 動畫只服務氛圍，不搶角色主動作。
