# 補給商店改版規格 v1

## 改版目標

補給商店要從「功能清單」改成「可每日回訪的補給櫃台」。它負責日常補給、一般收藏、扭蛋抽取與直購交換；不承擔節日主檔套裝的取得。

商店語氣：小魔女補給員、魔法櫃台、補給箱、星星兌換。畫面應偏工作台/櫃台式，而不是衣櫃頁或任務頁。

## 商店定位

- 每日補給：每天進來有一個小獎勵或限量交換。
- 一般收藏：常駐套裝、背景、稱號、頭像框、攻擊特效。
- 扭蛋抽取：用一般券/金券抽常駐收藏。
- 直購交換：用黃星/紫星換明確品項。
- 收藏管理：只保留輕量入口；主要裝備體驗仍在造型收藏頁。

## 不做的事

- 完整節日套裝不放普通商店。
- 完整節日套裝不放常駐扭蛋池。
- 商店不取代活動任務線。
- 商店不變成第二個造型收藏頁。
- 商店不把付費感做得太重；目前核心仍是遊戲內資源交換。

## 節日套裝規則

完整節日套裝，例如：

- `qixi_star_bridge_set`
- 未來春節、端午、中秋、聖誕/跨年完整套裝

取得來源只能是：

- 活動任務線
- 節日週獎勵
- 明確標示的復刻活動

商店可以販售或交換節日相關小物，但不可販售完整節日套裝。可接受例子：

- 節日稱號
- 節日頭像框
- 小型背景貼紙或特效
- 復刻活動入場券

不可接受例子：

- 直接用黃星/紫星買完整七夕套裝
- 把完整七夕套裝放進普通扭蛋池
- 活動還沒開放就讓一般商店永久取得節日套裝

## 新版資訊架構

### 1. 補給櫃台

首頁首屏上方區塊。

內容：

- 小魔女補給員或補給箱主視覺。
- 今日一句話，例如「今天也來補充一點勇氣吧」。
- 玩家持有資源：黃星、紫星、一般券、金券。
- 目前刷新時間或今日補給狀態。

用途：

- 讓商店像一個遊戲地點。
- 讓玩家快速知道今天能不能領/換。

### 2. 今日補給

每天回訪的核心。

可做項目：

- 免費每日補給：每天一次。
- 今日限量交換：每天 1-3 個品項。
- 週限量交換：每週刷新 3-5 個品項。

可給獎勵：

- 黃星少量
- 一般扭蛋券碎片或一般券
- 少量 EXP
- 今日攻擊小加成
- 預算護盾或超支緩衝道具
- 低稀有度稱號/頭像框碎片

每日補給不應直接給完整節日套裝。

### 3. 當期主打

展示目前商店可取得的一般內容。

可展示：

- 常駐一般套裝，例如薄荷補給套裝
- 新增常駐背景
- 新稱號或頭像框
- 當週推薦補給箱

不可展示：

- 完整節日套裝，除非它是復刻活動入口，且取得仍走活動規則。

### 4. 扭蛋機

保留一般補給箱與金色補給箱，但 UI 要更像遊戲機台。

需要顯示：

- 使用資源：一般券/金券。
- 可抽內容分類：特效、稱號、頭像框、常駐套裝。
- 機率：R / SR / SSR。
- 保底或重複轉換規則。

建議規則：

- 一般券：常駐 R/SR 為主，少量 SSR。
- 金券：SR/SSR 機率較高。
- 重複 R：轉黃星 1。
- 重複 SR：轉黃星 2。
- 重複 SSR：轉紫星 1。

### 5. 直購架

用黃星/紫星明確交換，不抽。

黃星適合：

- 一般券
- 低稀有稱號
- 頭像框
- 常駐背景
- 初階套裝或套裝碎片
- 今日消耗型小道具

紫星適合：

- SR 常駐套裝
- 稀有稱號
- 稀有頭像框
- 稀有背景
- 金券或金券碎片

完整節日套裝不放直購架。

### 6. 收藏入口

商店可保留「最近取得」或「補給收藏」摘要，但完整裝備管理應導向造型收藏頁。

建議：

- 顯示最近 3-6 個取得物。
- 提供「前往造型」按鈕。
- 不在商店內做完整衣櫃篩選。

## UI 方向

版面順序：

1. 返回列與頁名：`補給商店`
2. 補給櫃台主視覺
3. 資源列
4. 今日補給
5. 當期主打
6. 扭蛋機 / 直購架 tabs
7. 收藏摘要

視覺要求：

- 使用商店背景與補給員/補給箱素材。
- 卡片可以存在，但不要像資料表。
- 每個購買/抽取動作都要清楚顯示成本。
- 已擁有品項要顯示「已擁有」或轉換補償。
- 不使用 emoji 作正式圖示。

## 第一階段實作建議

第一階段先重排現有資料，不新增太多新經濟系統。

1. 新增今日補給區，但先只做每日免費領一次。
2. 重排上方為補給櫃台 + 資源列。
3. 把原本 direct/gacha/collection 改成更清楚的「今日 / 扭蛋 / 直購」。
4. 七夕套裝不放商店，只保留在活動任務與造型收藏 dev 預覽。
5. 保留原本抽卡與直購邏輯，先改 UI 與資訊架構。

## 驗收清單

- 首屏能看懂這是補給商店，不是造型收藏頁。
- 玩家能一眼看到今日可領什麼。
- 玩家能一眼看到黃星、紫星、一般券、金券數量。
- 完整節日套裝不出現在普通商店販售或常駐扭蛋池。
- 扭蛋、直購、每日補給的用途不同，不互相混淆。
- 手機底部 dock 不遮擋主要操作。
